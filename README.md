# Bank of Banks

 Es un agrupador de bancos, donde se puede consultar bolsa de valores,fondos de inversio≈Ñ y todas las posiciones que el cliente tiene en el banco.

Es una SPA usando:

- Angular y Bootstrap 
  
## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
